# Code of Conduct

We adopt the [Contributor Covenant, version 2.1][cc].

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Summary

- Be welcoming, be kind, assume good faith.
- Disagree about ideas, never about people.
- No harassment, personal attacks, or exclusionary behaviour.
- Maintain professional conduct in all project spaces — code
  reviews, issues, PRs, Discord, Keybase, email.

## Reporting

Conduct violations are reported via the same private channels as
security vulnerabilities. See [SECURITY.md](SECURITY.md) for the
contact addresses — until a dedicated conduct mailbox exists,
`security@ratesengine.net` reaches the maintainers and will be
handled with discretion.

## Enforcement

Maintainers may remove, edit, or reject any contribution that
violates this code of conduct, and may temporarily or permanently
ban contributors for behaviours they deem inappropriate,
threatening, offensive, or harmful.
